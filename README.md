# 📦 Product Management System – RESTful API

A fully functional RESTful API built using Java and 
Spring Boot supporting complete CRUD operations for 
product data management.

## 🚀 Tech Stack
- **Backend:** Java, Spring Boot, Spring MVC
- **Database:** MySQL, Spring Data JPA, Hibernate
- **Tools:** Postman, Eclipse, VS Code
- **Architecture:** REST API, OOP, Design Patterns

## ✨ Features
- ✅ Complete CRUD Operations (Create, Read, Update, Delete)
- 🗄️ Spring Data JPA for database interactions
- 🧪 API tested with Postman
- 📐 Clean, modular and maintainable code
- 🔧 RESTful best practices followed

## 📌 API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/products | Get all products |
| GET | /api/products/{id} | Get product by ID |
| POST | /api/products | Add new product |
| PUT | /api/products/{id} | Update product |
| DELETE | /api/products/{id} | Delete product |

## ⚙️ How to Run
1. Clone the repository
2. Configure MySQL in `application.properties`
3. Run as Spring Boot Application
4. Test APIs using Postman
